# Acexams 📚 (in Development)

Acexams is designed to help students prepare for exams, schedule study sessions, reach out for help and tack progress efficiently.\
It provides **study materials, previous year question papers, quizzes,
mock exams, and task scheduling** to help students organize their
preparation and track progress.

The platform includes **secure authentication, protected routes, and
personalized dashboards** to create a structured study experience.

------------------------------------------------------------------------

# 🚀 Features

## 🔐 Authentication

-   **Login**
    -   Generates JWT token for authentication
-   **Sign Up**
    -   Creates a new user account
    -   Generates JWT token for secure sessions

------------------------------------------------------------------------

## 📊 Dashboard

Displays important user data including:

-   Study progress
-   Scheduled tasks
-   Quick access to study material
-   Personalized study insights

------------------------------------------------------------------------

## 📝 Task Management

Students can manage their preparation using a built-in task system.

Features: - Schedule study tasks - Edit tasks - Delete tasks - Organize
preparation efficiently

------------------------------------------------------------------------

## 📅 Calendar

A calendar interface that stores all scheduled tasks and allows students
to visualize their study plan.

------------------------------------------------------------------------

## 📚 Study Material

The platform provides curated study resources\
- Previous Year Question Papers
- Sample Paper
- Mock exams
- Quiz(s)

------------------------------------------------------------------------

## 📄 Previous Year Question Papers

Students can access **previous exam papers** to practice and improve
exam preparation.

------------------------------------------------------------------------

## 🧠 Mock Exams

Mock exams simulate real test environments to help students evaluate
their preparation.

------------------------------------------------------------------------

## ❓ Quiz System

Interactive quizzes allow students to:

-   Test their knowledge
-   Identify weak areas
-   Improve topic understanding


------------------------------------------------------------------------

# 🧰 Tech Stack

## Frontend

-   JavaScript
-   ReactJS
-   TailwindCSS

## Backend

-   Node.js
-   Express.js

## Database

-   MongoDB

## Libraries & Tools

-   **JWT** -- Authentication and secure session handling
-   **bcryptjs** -- Password hashing and encryption
-   **mongoose** -- MongoDB object modeling

------------------------------------------------------------------------

# 🏗️ Architecture

Acexams is built using the **MERN stack**:

-   **MongoDB** -- Database
-   **Express.js** -- Backend framework
-   **React.js** -- Frontend framework
-   **Node.js** -- Runtime environment

This architecture allows the application to be **scalable, modular, and
efficient for handling user data and study resources**.
